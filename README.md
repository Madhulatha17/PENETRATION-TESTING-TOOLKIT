## Description

A Python-based modular penetration testing toolkit designed for security assessment and network reconnaissance in authorized environments.

## Features

* Port Scanner
* DNS Information Lookup
* HTTP Header Analyzer
* Modular Architecture
* Interactive Menu System

## Technologies Used

* Python
* Socket Programming
* Requests
* DNSPython

## Installation

pip install -r requirements.txt

## Execution

python main.py

## Modules

### Port Scanner

Scans common TCP ports on a target host.

### HTTP Header Analyzer

Retrieves and displays website security-related headers.

### DNS Lookup

Retrieves DNS A records for a domain.

## Deliverable

A Python-based modular toolkit with detailed documentation for authorized penetration testing and security assessment activities.
